1. Activate Custom Music
    1. Settings > Audio > (Scroll to bottom)
    2. Set "Radio Music Source" to "Default and Custom" or just "Custom"
    3. Click "Open Music Folder" to access the folder itself
2. Download Spotify Playlist(s)
    1. Locate playlist in Spotify
    2. Click three dot menu
    3. Click Share
    4. Click "Copy Link"
    5. Access [SpotifyDown](https://spotifydown.com/) to download playlist
        1. Access [www.spotifydown.com](https://spotifydown.com/)
        2. Paste playlist link
        3. Select Download
        4. Wait for .zip file to be downloaded (do not close tab or things won't download, or the world sets ablaze, I'm not sure...)
        5. Extract to Pacific Drive Custom Music Folder
